# Application - Workout Tracker (MVP)

## Core Problem
What problem does your application solve?

The Workout Tracker application is a tool for monitoring and planning workouts.
It will enable users to track their training progress and plan new workout sessions.

## Minimum Viable Product (MVP) Features
What is included in the MVP?

- Exercise list
- Adding new exercises to the current workout
- Saving a set of exercises as a workout template (e.g., Leg and Back workout)
- Tracking workout progress
- Saving completed workouts to history
- List of exercises in the current workout with fields for recording reps, sets, and weight
- Statistics summary after every workout
- Access to statistics in workout history

## What is NOT included in the MVP
What is out of scope for the MVP?
- Subscriptions
- Ads
- Onboarding experience
- Social features
- Premium features

## Success Criteria
What goal do I want to achieve?
- Ability to track workout progress
- Friendly and intuitive workout visualization
- Motivation to train
